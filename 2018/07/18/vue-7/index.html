脚手架里包含的文件夹

  • src文件夹:

    • assets图片
    • components所有的组件
    • router路由
    • App.vue主组件
    • Main.js全局配置
    • components所有内容渲染到App,vue主组件中，之后再main.js中进行挂在，然后最终渲染到index.html中

  • 选取DOM

    • 不推荐let box= document.选取DOM

    • 使用时要在方法下 this.$refs.自己取的要被选取的DOM名,nextTick:在vue方法中去改变data下的值时，后续的操作不能够直接拿到变动后的值，拿到的是之前data下声明的初始值,如果想拿到方法变动过之后的值时，需要使用$nextTick(()=>{})

  • 兄弟组件传值 childA—->父组件—–>childB

    • 先在chailda中定义自定义方法，把值传递给父组件

    • 父组件中使用props绑定的方式把childA传过来的值，传递给childB,childB使用props接收父组件传递过来的值

  • 路由配置

    • mode:’history’ 把路径中的#去除

    • path: ‘*’, *一般只做404页面使用，会匹配所有路径，但已有的路径不会被覆盖

    • 导航守卫：保护页面，进行权限拦截，没有权限不能查看

      • 全局守卫：写在main.js中，进行全部页面的拦截
        前置守卫：router.beforeEach((to, from, next) => {
        to进入的页面,from离开的页面,next()通行证，允许进入
        next()中可以传入path路径和true允许 false不允许 node express
        }
        后置钩子：router.afterEach((to, from)=>{没有next参数})
