列表渲染

  • 遍历对象：遍历对象一个值是val，两个值第一个val，第二是key三个值第一个val，第二个是key，第三个是index

  • 遍历数组：遍历数组只有两个值 item index，你也可以用 of 替代 in 作为分隔符，因为它更接近 JavaScript 迭代器的语法

  • key值管理:只进行遍历但不操作数据时可以不加key,如果使用key需要使用 v-bind绑定,key必须是唯一,一般情况下index当做key处理,不使用key值时进行数据操作会把DOM节点就地复用，如果使用key会把DOM节点摧毁重建，建议尽可能在使用 v-for 时提供 key attribute，除非遍历输出的 DOM 内容非常简单，或者是刻意依赖默认行为以获取性能上的提升

  • 向Vue实例化下追加数据:

    • 数组追加:Vue.set(数组,下标,添加的值)
    • Vue实例化对象.$set(数组,下标,添加的值)
    • 对象追加：Vue.set(对象,key,val)
    • Vue实例化对象.$set(数对象,key,val)

事件

  • event事件对象,methods下的方法默认是自带event事件对象的

  • 修饰符

    • 事件.stop阻止冒泡
    • 事件.prevent 阻止默认行为
    • 事件.self 事件只作用于本元素
    • 事件.once 事件只触发一次
    • 事件.capture 把冒泡事件改成捕获事件
    • 事件修饰符可以连写 <a :href="url" @click.prevent.once>百度</a>

  • 键盘修饰符: keydown.键码值或keydown.a enter delete

  • 事件和按键可以结合使用：@click.shift

  • @keydown.ctrl.exact=”fn” exact 修饰符允许你控制由精确的系统修饰符组合触发的事件。 精确匹配， 只有当前给定的按键生效，组合键不生效

  • mouseup mousedown 不区分左中右键
